# ts-toc-sort-imports README

Turns 

```js
import styles from './style.scss';
import React from 'react';
import { Link } from 'react-router-dom';
```

Into 

```js
import { Link } from 'react-router-dom';
import styles from './style.scss';
import React from 'react';
```

on file save!
